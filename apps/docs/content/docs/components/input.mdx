## Slots

- **base**: Input wrapper, trata alinhamento, posicionamento e aparência geral.
- **label**: Label do input, é o que é exibido acima, dentro ou à esquerda do input.
- **mainWrapper**: Envolve o `inputWrapper` quando a posição é `outside` / `outside-left`.
- **inputWrapper**: Envolve o `Label` (quando está dentro) e o `innerWrapper`.
- **innerWrapper**: Envolve o `input`, o `startContent` e o `endContent`.
- **input**: O elemento input.
- **clearButton**: O botão de limpar, está no final do input.
- **helperWrapper**: Envolve a `description` e a `errorMessage`.
- **description**: A descrição do input.
- **errorMessage**: A mensagem de erro do input.

## Data Atributos

  `Input` possui os seguintes atributos no elemento `base`:

- **data-invalid**:
  Quando o input é inválido. Baseado na propriedade `isInvalid`.
- **data-required**:
  Quando o input é obrigatório. Baseado na propriedade `isRequired`.
- **data-readonly**:
  Quando o input é somente leitura. Baseado na propriedade `isReadOnly`.
- **data-hover**:
  Quando o input está em hover. Baseado em [useHover](https://react-spectrum.adobe.com/react-aria/useHover.html).
- **data-focus**:
  Quando o input está sendo focus. Baseado em [useFocusRing](https://react-spectrum.adobe.com/react-aria/useFocusRing.html).
- **data-focus-within**:
  Quando o input está sendo focus ou algum de seus filhos. Baseado em [useFocusWithin](https://react-spectrum.adobe.com/react-aria/useFocusWithin.html).
- **data-focus-visible**:
  Quando o input está sendo focus com o teclado. Baseado em [useFocusRing](https://react-spectrum.adobe.com/react-aria/useFocusRing.html).
- **data-disabled**:
  Quando o input está desabilitado. Baseado na propriedade `isDisabled`.

## Acessibilidade

- Construído com um elemento `<input>` nativo.
- Suporte visual e ARIA para rotulagem.
- Suporte para eventos de mudança, área de transferência, composição, seleção e entrada.
- Estados obrigatórios e inválidos expostos à tecnologia assistiva via ARIA.
- Suporte para texto de ajuda de descrição e mensagem de erro vinculados ao input via ARIA.

## API

### Propriedades do Input

| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- | --------- |
| children         | `ReactNode`                                                                                                                                                            | O conteúdo do input.                                                                                             | -         |
| variant          | `flat` \| `bordered` \| `faded` \| `underlined`                                                                                                                        | A variante do input.                                                                                             | `flat`    |
| color            | `default` \| `primary` \| `secondary` \| `success` \| `warning` \| `danger`                                                                                            | A cor do input.                                                                                               | `default` |
| size             | `sm` \| `md` \| `lg`                                                                                                                                                   | O tamanho do input.                                                                                                | `md`      |
| radius           | `none` \| `sm` \| `md` \| `lg` \| `full`                                                                                                                               | O raio do input.                                                                                              | -         |
| label            | `ReactNode`                                                                                                                                                            | O conteúdo a ser exibido como rótulo.                                                                                  | -         |
| value            | `string`                                                                                                                                                               | O valor atual do input (controlado).                                                                          | -         |
| defaultValue     | `string`                                                                                                                                                               | O valor padrão do input (não controlado).                                                                        | -         |
| placeholder      | `string`                                                                                                                                                               | O espaço reservado do input.                                                                                         | -         |
| description      | `ReactNode`                                                                                                                                                            | Uma descrição para o input. Fornece uma dica, como requisitos específicos sobre o que escolher.                        | -         |
| errorMessage     | `ReactNode`                                                                                                                                                            | Uma mensagem de erro para o input.                                                                                       | -         |
| startContent     | `ReactNode`                                                                                                                                                            | Elemento a ser renderizado no lado esquerdo do input.                                                                 | -         |
| endContent       | `ReactNode`                                                                                                                                                            | Elemento a ser renderizado no lado direito do input.                                                                | -         |
| labelPlacement   | `inside` \| `outside` \| `outside-left`                                                                                                                                | A posição do rótulo.                                                                                            | `inside`  |
| fullWidth        | `boolean`                                                                                                                                                              | Se o input deve ocupar a largura de seu pai.                                                             | `true`    |
| isClearable      | `boolean`                                                                                                                                                              | Se o input deve ter um botão de limpar.                                                                         | `false`   |
| isRequired       | `boolean`                                                                                                                                                              | Se a entrada do usuário é necessária no input antes da submissão do formulário.                                                   | `false`   |
| isReadOnly       | `boolean`                                                                                                                                                              | Se o input pode ser selecionado, mas não alterado pelo usuário.                                                        |           |
| isDisabled       | `boolean`                                                                                                                                                              | Se o input está desativado.                                                                                        | `false`   |
| isInvalid        | `boolean`                                                                                                                                                              | Se o input é inválido.                                                                                         | `false`   |
| baseRef          | `RefObject<HTMLDivElement>`                                                                                                                                            | A referência ao elemento base.                                                                                          | -         |
| validationState  | `valid` \| `invalid`                                                                                                                                                   | Se o input deve exibir sua estilização visual "válido" ou "inválido". (**Depreciado**) use **isInvalid** em vez disso. | -         |
| disableAnimation | `boolean`                                                                                                                                                              | Se o input deve ser animado.                                                                                 | `false`   |
| classNames       | `Record<"base"｜ "label"｜ "inputWrapper"｜ "innerWrapper"｜ "mainWrapper" ｜ "input" ｜ "clearButton" ｜ "helperWrapper" ｜ "description" ｜ "errorMessage", string>` | Permite definir nomes de classes personalizados para os slots do input.                                                              | -         |

### Eventos do Input

| Atributo      | Tipo                                  | Descrição                                                                                                                          |
| ------------- | ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| onChange      | `React.ChangeEvent<HTMLInputElement>` | Manipulador chamado quando o valor do elemento muda. Você pode obter o novo valor acessando `event.target.value` (string).       |
| onValueChange | `(value: string) => void`             | Manipulador chamado quando o valor do elemento muda.                                                                                |
| onClear       | `() => void`                          | Manipulador chamado quando o botão de limpar é clicado.                                                                           |
