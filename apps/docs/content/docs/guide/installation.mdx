# Instalação

Requirements:

- [React 18](https://reactjs.org/) ou superior
- [Tailwind CSS 3](https://tailwindcss.com/) ou superior
- [Framer Motion 4](https://www.framer.com/motion/) ou superior

---


Para usar o VhsysUI em seu projeto, você precisa seguir os seguintes passos:

## Instalação Global

A maneira mais fácil de começar com o VhsysUI é usar a instalação global. Isso significa
que todos os componentes são importados de um único pacote.

Siga os passos abaixo para instalar todos os componentes do VhsysUI:

<Steps>

### Instalar Pacotes

Para instalar o VhsysUI, execute um dos seguintes comandos no seu terminal:

<PackageManagers
  commands={{
    npm: "npm i @vhsys-ui/react framer-motion",
    yarn: "yarn add @vhsys-ui/react framer-motion",
    pnpm: "pnpm add @vhsys-ui/react framer-motion",
  }}
/>

### Configuração do Tailwind CSS

O VhsysUI é construído em cima do Tailwind CSS, então você precisa instalar o Tailwind CSS primeiro. Você pode seguir o guia de [instalação oficial](https://tailwindcss.com/docs/installation) para instalar o Tailwind CSS. Em seguida, você precisa adicionar o seguinte código ao seu arquivo `tailwind.config.js`:

```js {8,13-14}
// tailwind.config.js
const {vhsysui} = require("@vhsys-ui/react");

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    // ...
    "./node_modules/@vhsys-ui/theme/dist/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  darkMode: "class",
  plugins: [vhsysui()],
};
```

### Configurando nosso Provider

É essencial adicionar o `VhsysUIProvider` na `raiz` da sua aplicação.

```jsx
import * as React from "react";

// 1. import `VhsysUIProvider` componente
import {VhsysUIProvider} from "@vhsys-ui/react";

function App() {
  // 2. Envolver o VhsysUIProvider na raiz da sua aplicação
  return (
    <VhsysUIProvider>
      <YourApplication />
    </VhsysUIProvider>
  );
}
```

### Configurar pnpm (opcional)

Se você estiver usando o pnpm, precisa adicionar o seguinte código ao seu arquivo `.npmrc`:

```bash
public-hoist-pattern[]=*@vhsys-ui/*
```

Após modificar o arquivo `.npmrc`, você precisa executar `pnpm install` novamente para garantir que as dependências sejam instaladas corretamente.

</Steps>
